Central Processing Unit

le proceseur de l'ordinateur est le composant qui réalise tous les calculs.

C'est le cerveau de l'ordinateur. Il calcule mais ne réalise pas l'interphase graphique: Le CPU envoie les informations à afficher à l'écran au GPU (carte graphique ) qui gére l'affichage.

Photo CPU

Il y a plusieurs informations importantes qui caractérise la puissance du CPU:

  1. Son nombre de bite: c'est le nombre de chiffre que le CPU est capable de gérer en même temps.
  2. Sa frequence: c'est le nombre de calculs que le CPU peut executer à la fois.
  3. Le nombre de cœur: c'est le nombre de proceseur conmposant le CPU.
  4. Sont socket: le format du processeur pour communiquer avec la carte mère
  5. Le refroidissement nécessaire car un CPU chauffe toujours plus ou moins.

Watercooling: exemple de systéme de refroidissement.
