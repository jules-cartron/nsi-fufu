Central Processing Unit

le processeur de l'ordinateur est le composant qui réalise tous les calculs.

C'est le cerveau de l'ordinateur. Il calcule mais ne réalise pas l'interface graphique: Le CPU envoie les informations à afficher à l'écran au GPU (carte graphique ) qui gére l'affichage.

Photo CPU

Il y a plusieurs informations importantes qui caractérisent la puissance du CPU:

  1. Son nombre de bite: c'est le nombre de chiffres que le CPU est capable de gérer en même temps.
  2. Sa frequence: c'est le nombre de calculs que le CPU peut executer à la fois.
  3. Le nombre de cœur: c'est le nombre de processeurs composants le CPU.
  4. Son socket: le format du processeur pour communiquer avec la carte mère.
  5. Le refroidissement nécessaire car un CPU chauffe toujours plus ou moins.

Watercooling: exemple de système de refroidissement.

Structure

Un CPU est constitué de plusieurs puces electroniques:

Shemas CPU

Une mémoire cache: elle sotcke les variables qui sont en train d'être calculées.

Le gestionnaire de méxmoire: il gère la répartition des variables dans la mémoire cache.

Puis les bus PCI: ils s'occupent de la gestion des informations qui rentrent et qui sortent du processeur.

Toutes ces parties sont gravées sur une puce en silicium.
